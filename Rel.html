type HashPair<X, Y> = (X -> Hash.Hash, Y -> Hash.Hash)

type EqualPair<X, Y> = ((X, X) -> Bool, (Y, Y) -> Bool)

type Rel<X, Y> = { forw : Trie.Trie2D<X, Y, ()>; back : Trie.Trie2D<Y, X, ()>; hash : HashPair<X, Y>; equal : EqualPair<X, Y> }

Relation between X's and Y's.

Uses two (related) hash tries, for the edges in each direction. Holds the hash and equal functions for the tries.

type RelShared<X, Y> = { forw : Trie.Trie2D<X, Y, ()> }

Relation between X's and Y's.

Shared type (no hash or equal functions).
